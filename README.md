# Stopwatch Tracker

A simple, elegant web-based stopwatch with an automatic cycle counter. Every time you complete a start-stop cycle, the counter automatically increments by 1.

## Features

✨ **Stopwatch** - Precise time tracking with millisecond accuracy
📊 **Automatic Counter** - Increments by 1 with each completed cycle
📈 **Cycle History** - Keep track of all your cycle times with timestamps
🎨 **Beautiful UI** - Modern, responsive design that works on all devices
⚡ **Real-time Display** - Live time updates as you track

## How to Use

1. **Start the timer** - Click the "Start" button to begin the stopwatch
2. **Stop the timer** - Click the "Stop" button to end the cycle
   - The counter automatically increases by 1
   - Your cycle time is added to the history
3. **Reset** - Click "Reset" to clear the timer and start fresh
   - Note: Resetting does NOT clear the counter or history

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - Stopwatch and counter logic

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start tracking your cycles!

## Display Format

- **Time Display**: `MM:SS:MS` (Minutes:Seconds:Centiseconds)
- **Counter**: Shows total number of completed cycles
- **History**: Lists all cycles with times and timestamps

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
